# Portfolio website

DESCRIPTION:
This is a Portfolio website of Kumarajeeva Elavarasan which comprises of details about his projects, education, skills and contact. This is a fully responsive website developed using React, SCSS, Framer motion, Sanity.io. 

INSTALLATION AND RUNNING:
* Clone the github repository and store it in your device.
* Download the latest versions of Node.js and Hyper command terminal.
* Download a text editor like Atom or Visual Studio Code.
* Go to the directory folder in Hyper and install npm package manager.
* Run the command "npm install".
* This will install the required packages and dependencies.
* Install "@sanity/cli" package and run the command "sanity init".
* Run the command "sanity manage" to open the manager.
* Copy the project id, token, apiversion and dataset and paste these in client.js file.
* Now create the required schemas like "skills, projects, education" with required fields.
* Now run the command "sanity start" to open it.
* Fill all the schemas with required data and images to be populated in the portfolio website.
* Now head to Hyper terminal, head to the project directory and run "npm start" command to run the project.
